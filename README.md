# Java Quiz Maker

Проєкт на Java: програма для створення вікторин з CSV та JSON запитань.  
Має два режими:  

- **Студент** – проходить вікторину, результат надсилається на email.  
- **Адміністратор** – переглядає, додає, редагує та видаляє питання у CSV та JSON файлах.  

## Особливості
- Підтримка CSV та JSON форматів для бази запитань.
- Випадкове формування вікторини.
- Перевірка правильності введення даних.
- Надсилання результатів на email.
  
## Як запускати
1. Скомпілювати всі файли:
```bash
javac -cp "lib/*:src/main/java" src/main/java/scr/*.java
```
2. Запустити програму:
```bash
java -cp "lib/*:src/main/java" scr.Main
```
